# **wall·i·draw - Your creative wallpaper app**

It uses the following tools/libraries:

## **Tech-Stack**

- React
- React Router
- Styled Components
- Vercel
- Canvas
- Storybook

---

## **Install**

Clone the repository

Install all dependencies and packages with:

```shell
npm install
```

Start project in dev mode:

```shell
npm start
```
